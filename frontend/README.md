# LinqB2B Frontend

The merchant dashboard and interface for Linq B2B.

## Tech Stack

- **Framework**: [Vite](https://vitejs.dev/)
- **Library**: [React](https://reactjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Configuration

Environment variables can be configured in a `.env` file (see `.env.example` if available, or check `src/` for required variables).
